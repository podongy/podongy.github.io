* Lambda function - \arg -> func 
 ex) (\x -> x * 2) 2   (=4) 
* Find type - :t expression
 ex) :t True   (True :: Bool)
 ex) :t (&&)   ((&&) :: Bool -> Bool -> Bool)
* List insertion - x : [] 
 ex) 1 : [] (=[1])
* Either a b 
* Maybe - True / False / Nothing
* type K = double / newtype K = K double
* data Bool = False | True
